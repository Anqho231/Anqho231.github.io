DIY Photoshop: Donut Shop Art Tool

My DIY Photoshop Drawing Tool is like a bakery setting drawing game where artists can draw whatever desserts they like. I drew donuts using circles as dough-like paint strokes, jelly, lines for sprinkles, and Oreo or chocolate cookie bits using ellipses. Each sprinkles are color coded different with blue, green, and yellow. The purpose is to draw desserts as a therapeutic resource and to imagine yourself as a baker in a bakery. This time, I included red strawberry sauce on key button 9, and light pink frosting for key button 0.
